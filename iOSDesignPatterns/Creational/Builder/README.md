# 建造者模式(`Builder`)

### 概念
将一个复杂对象的构建与它的表示分离，使得同样的构建过程可以创建不同的表示。

### 特点
分离属性，多步构建。

### 核心角色
产品类（Wine）
抽象建造者类（WineProductProtocol）
具体建造者类（Whisky,Brand）
指挥者类（SellWine）
