# 装饰模式(`Decrator`) 

### 概念
动态地给一个对象添加一些额外的职责。

### 特点
需要透明且动态地扩展类的功能时使用。
* 继承抽象组件
* 持有抽象组件引用

### 核心角色
抽象组件(StaticPhoneProtocol)
具体组件(StaticPhone)
抽象装饰者(DecPhone)
具体装饰者(DecHuweiPhone,DecApplePhone)
