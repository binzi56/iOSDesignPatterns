# 模板方法模式(`Template Method`)

### 概念
* 定义一个操作中的算法框架
* 将一些步骤延时到子类实现
* 子类可以不改变算法结构，可以重新定义算法的某些特定步骤

### 特点
将公有、重复、复杂的步骤设计为模版方法

### 核心角色
抽象类(GetUpBed)
具体模版实现类(JackGetUpBed)
